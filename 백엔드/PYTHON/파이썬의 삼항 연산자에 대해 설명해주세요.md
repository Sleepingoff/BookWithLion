</aside>

간단한 조건문을 한 줄로 표현할 수 있는 문법입니다. 삼항 연산자는 조건식을 나열하여 조건이 참인 경우와 거짓인 경우에 대해 서로 다른 값을 반환합니다. 일반적인 삼항 연산자의 구문은 다음과 같습니다:

`값1 if 조건식 else 값2`

다른 예시를 들어보면

```python

# 일반적인 조건문
if num % 2 == 0:
	print("짝수입니다")
else:
	print("홀수입니다")

# 삼항 연산자
print("짝수입니다" if num % 2 == 0 else "홀수입니다")
```

이처럼 표현할 수 있습니다.
